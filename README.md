# ✈️ British Airways Customer Review Dashboard (Tableau Project)

## 📊 Project Overview

This project is an interactive Tableau dashboard built to analyze and visualize customer reviews for British Airways. The goal is to identify key patterns in customer satisfaction, service quality, and overall travel experience using real-world review data.

The dashboard helps transform raw customer feedback into meaningful business insights that can support decision-making in airline service improvement and customer experience optimization.

## 🎯 Objectives

* Analyze customer review ratings across different service categories
* Identify trends in customer satisfaction and dissatisfaction
* Understand key factors affecting overall customer experience
* Visualize patterns in service quality performance
* Provide actionable insights for business improvement

---

## 📈 Dashboard Features

The Tableau dashboard includes:

* Overall customer rating analysis
* Service-wise breakdown (e.g., cabin service, food, boarding, etc.)
* Trend analysis of customer feedback
* Comparative insights across review categories
* Interactive filters for deeper exploration

## 📊 Key Insights
